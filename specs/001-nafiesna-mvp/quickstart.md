# Quickstart: NafieSna Mobile App MVP

## Setup

1. **Install Dependencies**:
   ```bash
   flutter pub get
   ```

2. **Run the App**:
   ```bash
   flutter run
   ```

3. **Verify Foundation**:
   - Ensure bottom navigation is functional.
   - Check that the theme is teal (#008080) and dark mode is active.

## Key Paths

- **Theme**: `lib/core/theme/`
- **Navigation**: `lib/core/navigation/`
- **Features**: `lib/features/`
- **Assets**: `assets/kaside.pdf`

## Verification Checklist

1. **Home Screen**: YouTube live component present (if online).
2. **Prayer Times**: Current daily times calculated and displayed.
3. **PDF Screen**: Continuous scroll is active for the local PDF.
4. **Content**: Religious days are visible.
