# UI Contract: NafieSna Mobile App MVP

## Navigation Contract
- **Home**: `@NafiEsna` live stream (at top) + Daily Hadith (scrollable below).
- **Prayer Times**: Current day's Fajr, Dhuhr, Asr, Maghrib, Isha (List view).
- **PDF**: Full-screen viewer for `assets/kaside.pdf`.
- **Content**: Grouped list view of upcoming religious days.

## Interaction Contract
- **Bottom Navigation**: Persistent bar with four icons (Home, Prayer, PDF, Content).
- **PDF View**: Drag vertically for continuous scroll; zoom via pinch.
- **Home View**: Tap the YouTube live section to open in the embedded player.

## Theme Contract
- **Dark Mode**: Mandatory and default.
- **Teal (#008080)**: Used for primary buttons, active nav icons, and app bar background.
- **Gold (#FFD700)**: Used for status indicators and high-importance highlights.
